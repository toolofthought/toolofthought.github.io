plot 패키지의 여러 타입을 알아봅시다. 공식 사이트에서 상세한 내용을 찾을 수 있습니다.

Area

기본 컬러코드를 사용하는 대신 http://colorbrewer2.org에서 가져온 컬러코드를 사용합니다.데이터의 차이나 변화량을 알아보고 싶을 때 도움이 되는 area타입입니다. matrix를 데이터 타입으로 다음 행과의 차이를 면적으로 표현합니다.

area타입은 2차원 그래프에서만 지원됩니다.

require 'plot'
MYCOLORS_z_ =: |. (".;._2) 0 : 0
255 247 236
254 232 200
253 212 158
253 187 132
252 141  89
239 101  72
215  48  31
179   0   0
127   0   0
)
BGC_z_ =: 255 255 248
'type area; color MYCOLORS; backcolor BGC' plot i. _9 20

응용: streamgraph

Surface

$$ z = x^2 - y^2. $$3차원 데이터를 그릴 때 도움이 되는 surface타입입니다. 3차원 그래프의 기본타입으로 지정되어 있습니다. 따로 타입을 정하지 않는다면 surface플롯을 그립니다.

surface타입은 3차원 그래프에서만 지원됩니다.

require 'plot numeric'
X =: Y =: steps _4 4 8
Z =: X -&*:/ Y
BGC_z_ =: 255 255 248 NB. plot 배경색, 꼭 locale을 _z_로 지정하세요.
'type surface; boxed 0; mesh 0; backcolor BGC' plot X ; Y ; Z
